## Open Source Launches

This section will grow as I publish frameworks, tools, and utilities.

### PSDM Framework

**Specification-first governance for AI-assisted software engineering.**

PSDM is a framework for governing software projects built with AI assistance. It helps teams and AI agents classify change risk, maintain specifications, document architecture decisions, review security/data/AI concerns, and apply delivery controls based on change impact.

**Status:** `1.0.0-beta.1`  
**Format:** Local CLI + GitHub Action  
**Stack:** Node.js · JSON contracts · GitHub Actions · governance templates  
**Repository:** [github.com/pablotech80/psdm-framework](https://github.com/pablotech80/psdm-framework)

> AI writes code. PSDM governs it.
